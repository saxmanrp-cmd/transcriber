# Sax Transcriber

A web app for transcribing saxophone notes in real time.

## How to use
Open `index.html` in Safari on iPhone, iPad, or Mac.

## Live site
https://rickparma.github.io/sax-transcriber/

## Editing
1. Click on `index.html` in this repo
2. Click the pencil (Edit) icon
3. Make your changes
4. Click "Commit changes"
5. Site updates automatically within 60 seconds
